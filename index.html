소개 프로젝트 연락처

프론트엔드 개발자 장성주

사용자 경험(UX)을 최우선으로 생각하며, React와 Next.js를 이용해 **효율적이고 성능 좋은** 웹 애플리케이션을 구축합니다.

주요 기술: React, Next.js, TypeScript, Tailwind CSS

주요 프로젝트 (3개)

1. 실시간 협업 툴 개발

Firebase와 WebSockets를 활용하여 문서 실시간 공동 편집 기능을 구현했습니다. (개발 기간: 3개월)

React Firebase Redux Styled Components

2. 대용량 데이터 시각화 대시보드

D3.js와 Vue.js를 이용해 100만 건 이상의 데이터를 처리하는 관리자 대시보드를 구축했습니다. (기여도: 70%)

Vue.js D3.js Node.js MongoDB

3. 개인 기술 블로그 (정적 사이트)

Next.js의 SSG 기능을 활용하여 마크다운 기반의 블로그를 제작하고 Vercel에 배포했습니다.

Next.js Markdown Vercel
